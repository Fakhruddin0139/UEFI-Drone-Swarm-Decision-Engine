# UEFI-GNSS-Telemetry-Parser
A bare-metal UEFI firmware application built in pure C (EDK2) that parses real-time quad-constellation GNSS data (GPS, GLONASS, BeiDou, Galileo) pre-OS. Features a Software-In-The-Loop (SITL) TCP simulation, zero floating-point math, and a Best Available Fix engine. The navigation foundation for an autonomous drone swarm.
